Changelog

Keep up with the development and changes of the app.

8 December 2025

  • Added a axes helper gizmo with the ability to snap to the axes.
  • Added tolerance setting for the connection clips.
  • Added corner magnet style for box and cutout models.
  • Added corner screw style for box and cutout models.
  • Added the ability to export a model without a stacking lip.

27 November 2025

  • Hover indication in the 3D environment when hovering element in sidepanel list.
  • Improved onboarding flow when entering the editor tool.
  • Fixed issue where the editor crashed when switching selected element.

25 November 2025

  • Complete design revamp of the application with a new look and more intuitive interactions.
  • Added tooltips to select settings.
  • Ability to show inches for the total measurements of your models.
  • Improved viewing experience with more intuitive camera movements and smart functions such as 'Fit to screen' button.

14 May 2025

  • Added the ability for users to save more models.

17 March 2025

  • Added translations for the application.

10 February 2025

  • Added the ability for logged in users to save up to 3 models in their account.
  • Developed the dashboard for managing saved models.
  • In addition to the 3 saved models, every 'file type' has a draft version that can be used as the base for new models.

22 January 2025

  • Added the ability to create a Gridfinity Generator account
  • Login with Google
  • Login with GitHub
  • Login with Discord
  • Login with email and password
  • With an account you can now save the settings of your current box or baseplate

31 December 2024

  • Fixed a bug where the outer walls were generated at half its intended thickness
  • Fixed a bug where a selected part was being deleted on backspace instead of clearing the selected input field

22 December 2024

  • Added a clip model to connect the baseplates. The model can be found under the 'connection clip' option when making a new model.
  • Baseplates can now be connected to each other using the new option "Connectable"

21 December 2024

  • Added the ability to customize the grid unit size, allowing to stray from the standard 42mm unit size

17 December 2024

  • Added controls to apply a max size to baseplates
  • Added controls to apply a max size to boxes and cutouts and cutting off the excess

11 December 2024

  • Added CAD kernel to allow for more control over more complex geometries
  • STEP file export
  • Fixed bug where control handles were included in the export
  • Custom magnet size
  • Adjustable outer wall thickness
  • Increased number of fractions per unit to place auxiliary parts from 6 to 12
  • Added backspace shortcut to delete the selected element
  • Duplication feature. Every auxiliary part can now be duplicated in their menu to the left
  • Ledges have been made larger
  • Scoop profile can be adjusted

3 November 2024

  • Improved drag and scale controls for ledges, scoops and walls
  • Changed the hover and selection colors for a clearer visual distinction

27 October 2024

  • Improved drag controls
  • Added toggle for snapping to grid
  • Fixed a bug where the length of the selected element could not be changed

26 October 2024

  • Major overhaul of the UI/UX of the app
  • Much improved mobile support in the viewer
  • Fixed a bug where the cutout lip and the walls were not aligned in Z
  • More control handles for all elements to make it easier to flip and rotate them
  • Property window to make it easier to understand the parameters of your different parts
  • More flexibility when controlling attributes via the property window as they do not conform to the strict snapping of the 3D-handles
  • Fixed a bug where the camera would break when holding the middle mouse button

21 October 2024

  • Added inner wall thickness to the box export
  • Added thin lip option to the box export

8 September 2024

  • Added baseplate magnets
  • Remade the voting UI for feature requests

7 September 2024

  • Added floor scoops with the same controls as the ledges
  • Fixed a bug where the corners of the ledges sometimes didn't fillet properly
  • Added new alternatives for feature requests
  • Added the ability to include baseplate magnets in the box and cutoutexport

6 September 2024

  • Added a new theme to the app
  • Increased the amount of places you can place the walls and ledges

5 September 2024

  • Added the ability to create ledges for stacking, gripping and to put labels on boxes
  • Added the changelog page

1 September 2024

  • Improved visual experience on mobile for the walls feature
  • Fixed bugs related to walls on mobile devices

22 August 2024

  • Improved camera orientation for better user control
  • Added the ability to vote for feature requests
  • Added the ability to suggest new features

8 August 2024

  • Introduced ambient occlusion feature called 'Beauty mode'

22 July 2024

  • Added cutout functionality to model generator
  • Fixed grid issues in floorplan

17 July 2024

  • Box generator
  • Basic Grid generator
  • STL file export
