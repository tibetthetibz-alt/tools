# Tools

A personal collection of client-side developer utilities. Everything runs entirely in your browser — no data is sent to any server.

## Tools

| Tool | Description |
|------|-------------|
| [Font Picker](fonts.html) | Browse 100+ Google Fonts by category, preview live, multi-select & copy `@import` |
| [HAR Separator](har-separator.html) | Drop a `.har` file, filter requests, inspect details, export cURL commands |
| [HTML → PDF](html-to-pdf.html) | Convert HTML to a downloadable PDF client-side with html2canvas + jsPDF |
| [Gmail Automator](gmail-automator.html) | Schedule and automate Gmail sends via OAuth2 — daily times or repeating intervals |

## Usage

Open `index.html` in your browser, or visit the GitHub Pages URL.

The site is deployed via GitHub Pages — no build step required, everything is plain HTML/CSS/JS.

## Tech

- Vanilla HTML, CSS, JavaScript
- No frameworks, no bundler
- Google Fonts (loaded from CDN)
- [jsPDF](https://github.com/parallax/jsPDF) + [html2canvas](https://html2canvas.hertzen.com/) for the PDF tool
